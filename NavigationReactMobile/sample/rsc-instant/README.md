# Navigation Mobile RSC Instant 
Try out instant Navigation Mobile in RSC's.

## Run
Once you've cloned the repository, you can install the dependencies and start the instant Navigation Mobile RSC example:

    npm install
    npm run build
    npm run start
	
Then visit http://localhost:3001/ in your browser to see instant Navigation Mobile in RSC's in action. Client `Scenes` with `Suspense` boundaries are shown instantly with fallbacks while child server `SceneViews` stream in.
