RSCs

The App router from Next.js introduced a lot of weird rules, for example, around partial page updates and access to the URL on the server. But they have nothing to do with RSCs (React Server Components) and everything to do with Next.js. There are actually very few rules when it comes to RSCs, as you can see with the Navigation router. All the Navigation router documentation, apart from the setup, works the same whether you use them or not.
